Learning Tracker - Overview

What it does
A lightweight desktop tool for logging study and project
sessions to a local SQLite database. One double-click to
start a session, another to stop. Automatically calculates
duration and logs date, time, topic, and notes. Built to
track progress across all my learning tracks over time.
Includes a built-in dashboard with live charts.

Status
Complete — actively in use.

GitHub
https://github.com/Memoria208/learning-tracker

Tech Stack
   * Python 3.14.2
   * SQLite (built into Python — no install needed)
   * matplotlib 3.x
   * tkinter (built-in Python GUI)

App Structure
   * tracker.py — main script, handles all logic
   * sessions.db — local SQLite database (not tracked in Git)
   * config.json — stores user's custom category list
   * session_state.json — temp file for active sessions
   * .gitignore — protects database and temp files

App Flow
   * Double-click shortcut — opens main menu
   * Main menu options:
     - Start Session — pick a category, timer starts
     - View Dashboard — opens live charts
     - Settings — manage categories
   * Double-click again while session is active — prompts
     to stop and log the session

Dashboard
   * Line chart — minutes logged over time
   * Pie chart — time broken down by topic
   * Charts group sessions by topic automatically
