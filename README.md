# Learning Tracker
A lightweight desktop tool for logging study and project sessions to Google Sheets — built to track progress across everything I'm working on.

## What it does
- One double-click to start a session — prompts for topic
- Another double-click to stop — calculates duration and logs everything
- Data goes straight to Google Sheets for tracking progress over time
- Built-in notes field for capturing what you learned or struggled with

## Why I built this
I'm a computer science student at College of Western Idaho, currently working across multiple tracks — including a hotsprings app, cybersecurity and privacy engineering, and general CS coursework. I built this tool to hold myself accountable across all of it and document my progress in a way that's visible and verifiable.

## Tech used
- Python 3
- Google Sheets API
- gspread
- tkinter (built-in Python GUI)

## Data tracked
Each session logs: data, start time, end time, duration (minutes), topic, and notes.

## Status
Active — used daily across all my current projects and learning tracks.
